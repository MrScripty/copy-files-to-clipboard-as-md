A cheap way to get a bunch of content pasted into a web chat interface like ChatGPT, Grok, Gemini etc...

- Drop the script in whatever directory you want to copy text based files from, then run. Everything will be on your clipboard.
- Use the recursive ersion if you also want everything from subdirectories copied.
- You can manualy add filetypes to parse or ask AI to edit if nessisary.
- Ask AI to convert to whatever OS you happen to be using if your not using Linux.
